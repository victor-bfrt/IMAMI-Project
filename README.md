# Breaking Bad Story

Ce projet a été réalisé dans le cadre du module d'informatique de CP1. 
Il s'agit d'un jeu textuel développé en Python. 
À chaque étape de l'histoire, le joueur doit faire des choix qui influencent le déroulement de l'aventure et la fin obtenue.
